## Getting Started

### Prerequisites
- A modern web browser

### Installation
1. Clone the repository
2. Open `index.html` in your browser

## Usage
1. Enter a task and click "Add"
2. Click the checkbox to mark tasks complete
3. Click "Delete" to remove tasks
4. Use filter buttons to view All, Active, or Completed tasks
